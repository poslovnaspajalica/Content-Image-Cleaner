== Description ==

**Content Image Cleaner** helps you clean up WordPress content and Media Library images for a selected date range.

Features:

- Remove featured images from posts/pages in a date range (by post date).
- Remove inline `<img>` tags and links to image files from post content (HTML cleanup).
- Optional PRO features may include deleting image attachments (disabled in this version).

This plugin is **images only** and does not delete PDFs or documents.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/content-image-cleaner/`
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to **Content Image Cleaner** in the WordPress admin menu.

== Frequently Asked Questions ==

= Will this delete PDFs or documents? =

No. All delete operations are limited to attachments with MIME type `image/*`.

= What does the "DELETE" confirmation do? =

It is a safety measure. For destructive actions you must type `DELETE` to confirm.
